# 阿里云安全组配置指南

## 问题
服务已正常运行，但无法从外部访问，原因是阿里云安全组未开放 5050 端口。

## 解决步骤

### 1. 登录阿里云控制台
访问：https://ecs.console.aliyun.com/

### 2. 找到你的ECS实例
- 实例ID: `i-bp18r6lfi33rtfs32100`
- 实例名称: `iZbp18r6lfi33rtfs32100Z`
- 区域: 华东1(杭州)

### 3. 配置安全组规则

1. 点击实例名称进入详情页
2. 点击 **"安全组"** 标签
3. 点击安全组ID进入安全组规则配置
4. 点击 **"添加安全组规则"**

### 4. 添加入方向规则

配置如下：
- **规则方向**: 入方向
- **授权策略**: 允许
- **优先级**: 1（默认）
- **协议类型**: 自定义TCP
- **端口范围**: 5050/5050
- **授权对象**: 0.0.0.0/0（允许所有IP访问）
  - 如果只需要特定IP访问，可以设置为你的IP地址
- **描述**: OCR系统服务端口

### 5. 保存并测试

保存后，等待几秒钟，然后访问：
- `http://8.136.59.48:5050`

## 验证

配置完成后，可以通过以下命令验证：

```bash
# 从本地测试连接
curl http://8.136.59.48:5050
```

如果返回HTML内容，说明配置成功。
